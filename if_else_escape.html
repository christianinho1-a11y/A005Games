How to Play

  1. Enter your name and pick a Level.
  2. Levels are tied to course rigor:
    • Level 1: Intro to Computer Science
    • Level 2: AP Computer Science Principles
    • Level 3: AP Computer Science A
    • Level 4: Honors Advanced Data Structures & Algorithms
    • Level 5: College-level CS
  3. Higher levels are worth more points per correct answer: L1 = 1 pt, L2 = 2 pts, L3 = 3 pts, L4 = 4 pts, L5 = 5 pts.
  4. Read the scenario and the if / else code. Choose what the code will actually do.
  5. You have 3 lives. A wrong answer costs 1 life.
  6. Each game uses up to 20 random questions from that level’s question bank, or until you lose all 3 lives.
